# Что смотреть в текущем проекте

## Frontend (визуальный прототип)

После запуска `frontend/Warehouse.Web` доступны страницы:

- `/` — панель управления
- `/auth` — авторизация и регистрация
- `/inventory` — остатки продукции
- `/requests` — заявки и выдача
- `/movements` — история перемещений
- `/employees` — сотрудники и отделы
- `/admin` — пользователи и роли

Важно: это UI-прототип. Данные пока статические, без подключения к API/БД.

## Backend (минимальный API)

После запуска `backend/src/Warehouse.Api` доступны:

- `/health` — проверка, что API работает
- `/api/warehouse/ping` — тестовый endpoint контроллера

## Диаграммы

Исходники диаграмм находятся в:

- `docs/assets/diagrams/Диаграмма_классов.puml`
