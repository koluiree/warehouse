# Документация проекта

В этой папке находятся инструкции для запуска и просмотра текущей версии проекта.

## Что есть сейчас

- `backend/src/Warehouse.Api` — минимальный ASP.NET Core Web API.
- `frontend/Warehouse.Web` — визуальный прототип интерфейса на Blazor WebAssembly.
- `docs/assets/diagrams` — актуальные `.puml` диаграммы.

## Файлы-гайды

- `docs/START.md` — быстрый старт: как запустить backend и frontend.
- `docs/VIEW.md` — что и где смотреть после запуска (маршруты страниц и API).
